Create a dat shortcut

dat_short will create a shortcut using HTTPS to a dat URL, with an optional HTTPS alternative. The dat URL will be served to Beaker Browser, the HTTPS one to all other browsers.

If you put the HTTP URL, say on Twitter, it will highlight as a normal URL (whereas a dat:// one will not).

HTTP URLs are not supported.
